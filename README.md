# BAR Lobby Reporting Mockup

Presentation-ready mockup for Beyond All Reason lobby reporting flows.

## Purpose

This mockup demonstrates how a player or spectator can report a user from:

- Play tab
- Watch tab
- Friends tab

## How To Open

1. Double-click `index.html`
2. Your default browser will open the mockup

## How To Use

### Main navigation

- `Play` opens the mockup lobby reporting flow
- `Watch` opens the replay reporting flow
- `Friends` opens the friends list reporting flow
- `News` contains a short presentation note
- `Library` contains a short presentation note

### Reporting flow

1. Right-click a player, spectator, friend, or a name in lobby chat
2. Click `Report User`
3. Choose `Chat Violation` or `Action Violation`
4. Select a recent game from the list
5. Optionally upload up to 3 `.png` or `.jpg` images
6. Enter the report reason
7. If it is a chat report, select one or more chat lines
8. Confirm the selected user or users
9. Submit the report and show the generated 7-digit report number

## Files

- `index.html`: app structure
- `styles.css`: BAR-style visual presentation
- `app.js`: mock data and interaction logic
- `assets/`: reference images used for the presentation
